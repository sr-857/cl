Build a complete, fully functional web application named Clarity Loop by converting an existing static HTML/CSS prototype into a working system. All pages, buttons, and UI elements must be interactive and connected. No placeholder or non-functional elements are allowed.

Objective

Transform the static UI into a realistic application simulation with:

Functional navigation across all pages
Dynamic data-driven rendering
Working user interactions
Persistent state using localStorage
Clear separation of components and logic
Technology Requirements (Mandatory)
React (with Vite)
React Router for navigation
React Hooks for state management
localStorage for persistence
Chart library (Recharts or Chart.js)
