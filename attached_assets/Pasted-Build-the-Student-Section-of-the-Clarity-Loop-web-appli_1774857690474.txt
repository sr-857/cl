Build the Student Section of the Clarity Loop web application using React. This section must be fully functional, data-driven, and interactive, with no static placeholders or non-working elements.

Objective

Create a complete student experience where a user can:

Browse and read story-based lessons
Interact with content (highlight, simplify, doubts)
Attempt quizzes
Track performance and progress
Continue learning based on weak areas
Tech Requirements
React (Vite)
React Router
React Hooks (useState, useEffect)
localStorage for persistence
Pages to Implement
Library Page
Reader Page
Student Dashboard
1. Library Page (Dynamic Story Listing)
Data Source

Create stories.json:

[
  {
    "id": 1,
    "title": "The Car Race and Velocity",
    "subject": "Physics",
    "story": "A race between two cars explains velocity...",
    "concept": "Velocity is speed with direction...",
    "keyPoints": ["Velocity has direction", "Speed is scalar"],
    "questions": [
      { "q": "Define velocity", "a": "Velocity is speed in a given direction" }
    ]
  }
]
Required Features
Load all stories dynamically
Display:
Title
Subject
Short preview
Add search and subject filter (basic logic)
On click → navigate to Reader page with story ID
